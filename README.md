# Kubernetes Guardian 🛡️

A deployment safety toolkit for Kubernetes.

## Overview

- Safety Gates
- Canary Analysis  
- Incident Prevention

## Installation

```bash
curl -L https://github.com/rohankapoor161/k8s-guardian/releases/download/v0.1.0/guardian-linux-amd64 -o guardian
chmod +x guardian
```

## Quick Start

```bash
guardian validate -f deployment.yaml
```
