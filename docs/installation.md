# Installation

## macOS

```bash
brew install rohankapoor161/tap/k8s-guardian
```

## Linux

Download the binary:
```bash
curl -sL https://github.com/rohankapoor161/k8s-guardian/releases/download/v0.1.0/guardian-linux-amd64 -o /usr/local/bin/guardian
chmod +x /usr/local/bin/guardian
```

## From Source

```bash
go install github.com/rohankapoor161/k8s-guardian/cmd/guardian@latest
```
